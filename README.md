# nontoxic_freeroom
an alt:V Freeroam resource

## installation
- Clone the repo into the resources folder
```bash
cd altv-server/resources
git clone https://github.com/Timo972/nontoxic_freeroom.git
```
- add the resource to the server.cfg
```cfg
...
resources: [
  nontoxic_freeroom
  ...
]
...
```
